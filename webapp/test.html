Testing Overview

This is an overview page of various ways to test the generated app during development.
Choose one of the access points below to launch the app as a standalone application, e.g. on a Tomcat server.

  • testFLP.html - start the app in the SAP Fiori launchpad sandbox (with the real backend server) - only details working with backend, search is implemented using mock data
  • testFLPService.html - start the app in the SAP Fiori launchpad sandbox (with local mock data)

    • Integration Tests

  • test/integration/opaTests.qunit.html - run all integration tests
  • test/integration/opaTests.qunit.html?module=Cross%20App%20Navigation - run Cross App Navigation integration tests
  • test/integration/opaTests.qunit.html?module=In%20App%20Navigation - run In App Navigation integration tests
  • test/integration/opaTests.qunit.html?module=Display%20Time%20Deposit - run Display Time Deposit integration tests
    • -->
  • test/integration/opaTests.qunit.html?module=Errors - run Error integration tests
  • test/integration/opaTests.qunit.html?module=Search%20Customer - run Search integration tests
  • test/integration/opaTests.qunit.html?module=Display - run Display integration tests
  • test/integration/opaTests.qunit.html?module=In%20App%20Navigation - run In App Navigation integration tests

    • Test Suite

  • test/testsuite.qunit.html - run all the tests
